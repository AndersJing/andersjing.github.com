Centos 6.5 安装python 3

  • Python
  • 2018-01-11

1.下载Python-3

#wget http://python.org/ftp/python/3.5.4/Python-3.5.4.tar.xz

2.解压

#xz -d Python-3.5.4.tar.xz
#tar -xvf Python-3.5.4.tar

3.更改工作目录

#cd Python-3.5.4

4.安装 

#./configure

#make all

#make install

#make clean

#make distclean

5.查看版本信息 

#/usr/local/bin/python3 -V

6.建立软连接

#ln -s /usr/local/bin/python3 /usr/bin/python3

7.重新检验Python 版本 

#python3 -V
